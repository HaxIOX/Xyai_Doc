# 套餐相关问题解答

### 1.套餐叠加

**新套餐临时替代旧套餐**

当您购买新套餐时，新套餐会立即生效暂时替代旧套餐，新套餐到期后，旧套餐自动恢复生效，继续使用剩余时长，您购买的所有时长都不会浪费。

> 您可以在**个人中心**->**订阅套餐**中查看当前生效套餐

### 2.是否可以中途升级套餐

可以随时购买新套餐。

新套餐会立即生效，旧套餐会在新套餐到期后自动恢复。

### 3.买错套餐

如果您不小心买错了套餐，可以联系站长进行退款，您需要提供的有

1. 用户名
2. 付款截图
3. 计算好的剩余价值（如果有的话）

然后耐心等待站长的回复。
