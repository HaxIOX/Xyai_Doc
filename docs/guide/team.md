# team自助邀请

## 自助绑定

在[xyai的小铺](https://newshop.xychatai.com/cat/recommend)购买卡密，然后在[team自助邀请](https://new.xychatai.com/pastel/#/custom/1)中兑换，绑定邮箱即可

**质保卡密**可以在翻车后进行自助换绑

![image-20251115145002140](assets/image-20251115145002140.png)

解绑后使用同一个兑换码重新绑定即可。

温馨提示：

>⚠️：兑换码的有效期为三天，为了让更多的佬可以可以用到，大家尽量不要囤兑换码

如果不小心忘记使用，可以联系站长进行退款，您需要提供

1. 用户名

2. 付款截图
3. 计算好的剩余价值（如果有的话）

然后耐心等待站长的回复即可。
