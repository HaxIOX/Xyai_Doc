## 🚨使用限制说明

为确保正常使用，请了解当前系统限制机制。本平台限制机制分为两类：

#### 平台自身限制（我们设置的）

- 同步官网限速，目前的配额是官网pro配额的一半，后续根据具体情况会有所提升。
- claude限速更改同步官网usage计费
- 每个用户分为5小时使用，7天使用限制，7天opus使用限制
- 计算方式为，你提问的时间点当前的账号的使用限额为出发点。提问结束为结束点。对应官网的账号的配额变化
- 如果出现多人共同使用则按照，多人加权平均算法，尽可能的公平公正。

#### Claude Pro 套餐限制（由 Claude 官方设定）

- Token 消耗计费：Claude 以 token 为单位计费，消息长短、内容复杂度都会影响消耗。
- 额度折算说明：平摊下来，约为 每5小时可使用45次，超过后需等待额度恢复。
- Deep Research 模式：此功能消耗额度较大，建议在账号额度“满血”状态下使用，或通过实际使用自行探索最佳触发方式。

## 📌使用提示

- 聊天记录上限与账号绑定：每个 Claude 官方账号有固定的聊天记录上限。即使切换“车辆”（账号），打开某条历史记录仍会绑定回原账号，若该账号记录已满，将无法继续对话。
- 延续对话建议操作： 使用“导出聊天记录”功能保存当前内容。 切换到一个未达记录上限的新账号（车辆）。 将导出的聊天记录作为附件上传，发送给 Claude，以继续原对话上下文。
